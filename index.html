Recursive Trees!

This was my first attempt at creating a recursive tree. You can regenerate the tree by pressing any key once on the webpage.

Recursive Trees 2!

As suggested by the title, this was my second attempt at generaging recursive trees. This one improves on the first attempt as follows:

  1. It builds the tree first, then gives you ways to change the way the tree is displayed via a set of sliders.
  2. You can regenerate a new tree by depressing any key on the keyboard whilst in the webpage.
